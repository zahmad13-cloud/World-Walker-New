#World Walker Hackathon Project

This is my hackathon project for BigThink × WorldLabs.  
It’s an AI-driven game where users enter a prompt, and a Marble world is generated from the location they choose.

All the user has to do is create an account on World Labs Marble and retrieve an API key. They then input this API key into the provided line on World Walker.
Then they can type in the description for the world they would like to enter.

The user will now be transported to a 3D game-like setting described by their prompt. They will be able to move around using keys and mouse.
They can then move towards a teleporter that will direct them to the world generated by the 3D prompt in marble.
This world can be used in Unity or any similar software to create cool scenes and more video games.

World Labs x Marble were a big help here.
